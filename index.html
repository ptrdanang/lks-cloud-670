Apa Itu Cloud Computing?

Cloud Computing atau Komputasi awan adalah teknologi yang menjadikan internet sebagai pusat pengelolaan data dan aplikasi. Pengguna deberikan akses untuk mengelola suber daya server, penyimpanan, dan perangkat lunak secara virtual tanpa harus memilki infrastruktur fisik sendiri. Hal ini memunjgkinkan efisien biaya, aksebilitas tinggi, dan kemudahan pengembangan aplikasi berskala besar.

Model Layanan Cloud

Cloud Computing umumnya dibagi menjadi tiga model layanan utama:

Iaas (infrastruktur)

Penyendia suber daya komputasi dasar seperti Virtual Machine (VM), jaringan, dan penyimpanan data.

Paas (Platfrom)

Menyediakan lingkungan bagi pengembang unutk membangun dan mengelola aplikasi tanpa mengurus infrastruktur.

Saas (Software)

Aplikai siap pakai yang dpat diakses langsung oleh pengguna melalui internet (browser).

Fitur Dinamis (LKS Requirement)
